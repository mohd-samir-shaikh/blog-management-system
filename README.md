# Blog Management System

A Laravel-based Blog Management System with authentication, blog CRUD operations, image uploads, AJAX category filtering, and responsive UI.

## Features

- User Registration & Login
- Create, Edit & Delete Blogs
- Blog Categories
- Image Upload
- AJAX Category Filter
- Search Blogs
- Pagination
- CKEditor Integration
- Responsive Design

## Tech Stack

- Laravel
- PHP
- Bootstrap 5
- jQuery AJAX
- SQLite (Render Deployment)
- MySQL (Local Development)
